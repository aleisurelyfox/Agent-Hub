# Agent-Hub

一个用于构建与集成智能 Agent 能力的仓库骨架。当前为初始版本，包含基础协作规范与 CI 工作流，便于后续扩展到 Node/Python 等技术栈。

## 功能概览
- 基础仓库结构与忽略规则
- Issue/PR 模板提升协作质量
- Node 与 Python 通用 CI 工作流
- CodeQL 安全扫描与依赖审计建议

## 目录结构（初始）
- `.github/`：工作流与模板
- `README.md`：项目说明
- `LICENSE`：开源许可
- `.gitignore`：忽略规则

## 后续规划
- 根据具体语言与框架补充 `src/`、测试目录与构建脚本
- 扩展工作流（构建、发布、镜像与包管理）
- 启用更全面的安全检查与代码质量工具

欢迎通过 Issue/PR 参与贡献。
